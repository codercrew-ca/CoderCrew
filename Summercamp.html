The micro:bit is a physical computing device for children. Using Microsoft's MakeCode web platform, children will be taught to program on micro:bit hardware. This enables children to understand how software and hardware work together.

Apart from this, children will learn to program LED patterns, sensors, and will be introduced to Bluetooth as well.
